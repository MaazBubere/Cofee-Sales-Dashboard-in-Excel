# Cofee-Sales-Dashboard-in-Excel

A dynamic and interactive **Coffee Sales Dashboard** built using **Microsoft Excel** to analyze coffee shop sales performance and business insights. This project helps track revenue, customer behavior, product performance, and sales trends through advanced Excel visualizations and reporting tools.

---

# 📌 Features

- 📊 Interactive Excel Dashboard
- ☕ Product-wise Sales Analysis
- 💰 Revenue & Profit Tracking
- 📈 Monthly & Yearly Sales Trends
- 🌍 Region-wise Sales Performance
- 🛒 Customer Purchase Insights
- 📉 KPI Metrics and Reporting
- 🎯 Slicers and Filters for Dynamic Analysis

---
